背景介绍

配置Hibernate时遇到的小问题
数据库 MySQL
数据库版本 8.0.18 MySQL Community Server - GPL

MySQL 8.0 以上版本的数据库连接有所不同:

使用8.0以上版本的驱动包 mysql-connector-java-8.0.xx.jar
Url = jdbc:mysql://localhost:3306/xp1911?useSSL=false&serverTimezone=UTC
Driver = com.mysql.cj.jdbc.Driver

因为URL中有 & 字符，在XML中使用时必须要转义

不合法的XML字符必须被替换为相应的实体。& 就是其中之一

<!-- 错误写法 -->
<property name="connection.url">
jdbc:mysql://localhost:3306/xp1911?useSSL=false&serverTimezone=UTC
</property>

<!-- 修改如下 -->
<property name="connection.url">
jdbc:mysql://localhost:3306/test?useSSL=false&amp;serverTimezone=UTC&amp;characterEncoding=UTF-8
</property>
